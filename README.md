# Indian Sign Language (ISL) Recognition System

This project implements a prototype system to recognize Indian Sign Language (ISL) gestures and convert them into text subtitles.

## Features
- Real-time hand gesture capture using webcam
- ISL alphabet and basic gesture recognition
- Subtitle generation from detected signs
- Modular Python-based implementation

## Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy
- TensorFlow / PyTorch (if used)

## Project Status

- Current scope: ISL alphabets and basic gestures
